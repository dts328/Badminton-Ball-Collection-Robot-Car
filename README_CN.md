# 羽毛球收集机器人小车

一套集视觉识别、自主导航与精准抓取于一体的智能羽毛球收集机器人系统。

## 系统概述

系统采用**边缘视觉-ROS2决策-嵌入式控制**三层架构，形成从感知到执行的完整闭环。

```
┌─────────────────────────────────────────────────────────────────┐
│                        系统总体架构                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ┌─────────────┐     UART      ┌─────────────┐    ROS2 DDS    │
│  │   RDK X5    │ ────────────→ │    ROS2     │ ←───────────→   │
│  │  (视觉识别) │   识别结果    │  (导航决策) │    话题通信     │
│  └─────────────┘               └──────┬──────┘                 │
│                                       │ UART                   │
│                                       ▼                        │
│                              ┌─────────────────┐               │
│                              │  STM32F407      │               │
│                              │  (底层控制)      │               │
│                              └────────┬────────┘               │
│                                       │ PWM/编码器             │
│                                       ▼                        │
│                              ┌─────────────────┐               │
│                              │  麦克纳姆底盘    │               │
│                              │  + 机械臂        │               │
│                              └─────────────────┘               │
└─────────────────────────────────────────────────────────────────┘
```

## 硬件配置

| 组件 | 型号 | 接口 |
|------|------|------|
| 主控 | STM32F407VET6 | - |
| 视觉计算 | 地平线RDK X5 | - |
| 激光雷达 | 乐动LD06 | /dev/ttyACM2 @ 230400 |
| 深度相机 | 奥比中光 | USB |
| 底盘 | 麦克纳姆轮×4 | /dev/ttyACM0 @ 115200 |
| 机械臂 | 5+1自由度 | /dev/ttyACM1 @ 115200 |

## 核心功能

### 视觉识别
- YOLOv8-Pose模型 + BPU推理
- 15帧滑窗时序投票机制
- 二分类检测：完好球(0) / 破损球(1)
- mAP50: 0.995

### 自主导航
- SLAM Toolbox建图
- AMCL定位
- Nav2路径规划 + TEB局部规划器
- 全向移动支持

### 运动控制
- 10ms控制周期 (100Hz)
- 增量式PID + 电压前馈补偿
- 麦克纳姆轮运动学解算
- 5+1自由度机械臂控制

## 快速开始

### STM32
1. 使用Keil MDK打开 `stm32/MDK-ARM/project.uvprojx`
2. 编译并烧录到STM32F407

### ROS2
```bash
cd ros2

# 构建工作空间
./build.sh

# 开始建图
./start_slam.sh

# 开始导航
./start_nav.sh

# 开始任务
./start_task.sh

# 停止所有
./stop_all.sh
```

## 通信协议

### 二进制协议 (ROS2 ↔ STM32)

**接收帧 (11字节)**:
```
[0x7B][Mode][Res][X_H][X_L][Y_H][Y_L][Z_H][Z_L][Chk][0x7D]
```

**发送帧 (26字节)**:
```
[0x7B][flags][encA][encB][encC][encD][accel][gyro][bat][Chk][0x7D]
```
